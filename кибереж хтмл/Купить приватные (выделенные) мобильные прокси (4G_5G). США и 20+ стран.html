                                    Инструкция: Генератор учетных данных

                                    Генератор учетных данных используется для создания логинов и паролей для подключения к резидентским ротационным прокси.

                                    Важно:

                                    IP-адрес вручную не указывается.

                                    IP автоматически назначается прокси-сервером на основе параметров, заданных в логине.

                                    Как работает генератор

                                    В логине указываются параметры подключения:

                                    • тип прокси (резидентский)
                                    • тип сессии
                                    • страна, регион, город (если применимо)

                                    Прокси-сервер анализирует логин и сам выбирает подходящий IP-адрес из пула.

                                    Протоколы и порты

                                    • HTTP / HTTPS — порт 5959
                                    • SOCKS5 — порт 9595

                                    Авторизация всегда по логину и паролю.

                                    Форматы учетных данных

                                    В генераторе доступны 3 формата вывода:

                                    1. IP:PORT:USERNAME:PASSWORD
                                    2. USERNAME:PASSWORD@IP:PORT
                                    3. PROTOCOL://USERNAME:PASSWORD@IP:PORT

                                    Формат выбирается в генераторе в зависимости от используемого софта.

                                    Типы сессий

                                    Доступны 3 типа сессий.

                                    1. Случайный IP

                                    • IP меняется при каждом подключении
                                    • Страна выбирается автоматически

                                    Логин:

                                    LOGIN-res-any

                                    Результат:

                                    Каждое новое подключение → новый IP

                                    2. Короткая сессия (до ~1 минуты)

                                    • Один IP закрепляется примерно на 1 минуту
                                    • Можно выбрать страну, регион и город

                                    Логин:

                                    LOGIN-res-any-sid-12345678

                                    sid — произвольное число.

                                    При смене sid будет назначен новый IP.

                                    Пример с геолокацией:

                                    LOGIN-res_sc-us_georgia_macon-sid-54683597

                                    Где:

                                    • us — страна
                                    • georgia — регион (штат)
                                    • macon — город

                                    3. Долгая сессия (до 6 часов)

                                    • Один IP сохраняется до 6 часов
                                    • Можно выбрать только страну

                                    Через генератор (рекомендуется)

                                    В генераторе:

                                    • выберите страну
                                    • включите «долгую сессию»

                                    Готовый логин будет сгенерирован автоматически.

                                    Ручное получение токена

                                    1. Первый запрос:

                                    curl -v -x http://LOGIN-resfix-us-nnid-0:PASSWORD@51.77.190.247:5959 https://ipv4.icanhazip.com

                                    2. В ответе:

                                    X-NN-LLS: 9d016e262509d3827293

                                    3. Используйте логин:

                                    LOGIN-resfix-us-nnid-9d016e262509d3827293

                                    Массовая генерация учетных данных

                                    Генератор позволяет:

                                    • создавать неограниченное количество логинов
                                    • использовать неограниченное число потоков
                                    • генерировать логины по странам и сессиям

                                    Если страна не указана — IP и страна выбираются автоматически.
